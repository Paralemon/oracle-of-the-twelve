# Oracle of the Twelve

A divination app cast with three twelve-sided dice. Shake the phone (or press
**Cast**) to stir the dice; hold still to let them fall. The three results are
read as a single astrological placement, and an AI oracle interprets the cast.

## Try it on your phone

**▶ https://oracle-of-the-twelve.vercel.app**

Open that link on any phone, anywhere — no install needed, and the **AI reading
works** (the key lives server-side on Vercel). On iOS, tap **Enter** then shake
to cast (HTTPS unlocks the motion sensor); you can also tap **Cast the Dice**.

**Install it like an app:** open the link, then
- **iPhone (Safari):** Share → *Add to Home Screen*
- **Android (Chrome):** ⋮ menu → *Install app* / *Add to Home screen*

It then launches full-screen from your home screen and works offline (the cast;
readings need a connection).

> A GitHub Pages mirror also exists at
> https://paralemon.github.io/oracle-of-the-twelve/ and now calls the Vercel
> backend for readings via CORS. The Vercel URL above is canonical.
> Override the backend for any build with `?oracle=https://your-backend/interpret`.

## The three dice

| Die | Body | Symbols |
| --- | --- | --- |
| 1 | white | planetary & lunar-node glyphs — ☉ ☽ ☿ ♀ ♂ ♃ ♄ ♅ ♆ ♇ ☊ ☋ |
| 2 | black | the twelve zodiac signs — ♈ … ♓ |
| 3 | grey | numbers 1–12, read as the astrological **house** |

The cast reads as: **planet (what) · sign (how) · house (where)**.

## Randomness

The tumble is driven by `crypto.getRandomValues` (OS entropy), not
`Math.random()` — the face that lands is decided by genuine chance.

## Running it

The app is static (Three.js + cannon-es via CDN, no build step), but the
interpretation feature needs a tiny backend so the Claude API key never
reaches the browser.

```sh
export ANTHROPIC_API_KEY=sk-ant-...   # required for readings
python3 server.py                     # serves http://localhost:8124
```

Without a key the app still casts the dice; the **Interpret** button just
reports that the oracle is silent. The backend uses the official Anthropic
Python SDK when installed (`pip install anthropic`) and falls back to a
dependency-free `urllib` call otherwise. The reading is generated by
`claude-opus-4-7` with adaptive thinking and prompt caching.

## Files

- `index.html` — UI, styling, the full-screen reading overlay
- `main.js` — dice geometry, physics, presentation, RNG, interpretation wiring
- `server.py` — static server + `/interpret` endpoint (holds the API key)
- `.claude/launch.json` — preview server config (`python3 server.py`, port 8124)
